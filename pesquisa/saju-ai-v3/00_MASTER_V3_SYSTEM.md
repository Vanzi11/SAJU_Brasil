# VISÃO GERAL

Este documento define a arquitetura central do sistema V3.

A V3 representa a transição de:
- “relatório de Saju”
para:
- framework autoral de análise comportamental orientado por IA.

O objetivo do sistema NÃO é:
- previsão mística,
- astrologia fatalista,
- prática religiosa.

O objetivo do sistema É:
- análise de padrões comportamentais,
- identificação de dinâmicas emocionais,
- interpretação simbólica aplicada,
- apoio ao autoconhecimento.

---
