# 31_PROMPT_ENGINEERING_SYSTEM
## Prompt Architecture & Cognitive Instruction System
## Version 1.0

Based on:
- Multi-Agent AI Systems
- Behavioral AI Design
- Narrative Psychology
- RAG Architectures
- Saju/BaZi Structural Methodology

---

# 1. VISÃO GERAL

O sistema NÃO deve depender:
- de prompts gigantes,
- prompts emocionais genéricos,
- instruções caóticas.

Ele deve utilizar:
# arquitetura modular de prompts.

---

# 2. CORE PRINCIPLE

Prompts devem funcionar como:
# sistemas cognitivos especializados.

Cada prompt:
- possui responsabilidade clara,
- escopo limitado,
- comportamento definido.

---

# 3. OBJETIVO

Criar:
- consistência,
- previsibilidade,
- profundidade,
- controle narrativo,
- precisão estrutural.

---

# 4. PROMPT HIERARCHY

A arquitetura deve seguir:

```text
System Prompt
↓
Agent Prompt
↓
Task Prompt
↓
Context Injection
↓
User Input
```

---

# 5. SYSTEM PROMPT

O System Prompt:
# governa toda a plataforma.

Ele define:
- filosofia,
- limites,
- tom,
- regras globais.

---

# 6. SYSTEM PROMPT RULES

O sistema deve:
- evitar fatalismo,
- evitar religião,
- evitar previsão extrema,
- priorizar psicologia,
- priorizar estrutura.

---

# 7. AGENT PROMPTS

Cada agente deve possuir:
- identidade cognitiva,
- função específica,
- limitações claras.

---

# 8. CALCULATION AGENT PROMPT

Deve:
- operar matematicamente,
- evitar narrativa,
- evitar interpretação psicológica.

---

# 9. STRUCTURE AGENT PROMPT

Deve:
- interpretar sazonalidade,
- avaliar Useful God,
- analisar fluxo energético,
- validar estrutura.

---

# 10. PSYCHOLOGY AGENT PROMPT

Deve:
- traduzir estruturas em comportamento,
- usar linguagem moderna,
- evitar exagero emocional.

---

# 11. RELATIONSHIP AGENT PROMPT

Deve:
- analisar dinâmica emocional,
- avaliar comunicação,
- interpretar estilos de apego.

---

# 12. NARRATIVE AGENT PROMPT

Deve:
- gerar profundidade,
- criar retenção emocional,
- manter elegância narrativa.

---

# 13. LOCALIZATION AGENT PROMPT

Deve:
- adaptar tom cultural,
- localizar emoção,
- evitar tradução literal.

---

# 14. SAFETY PROMPT

A IA deve:
- evitar linguagem perigosa,
- evitar previsões médicas,
- evitar manipulação emocional,
- evitar dependência psicológica.

---

# 15. RAG INJECTION SYSTEM

Prompts devem receber:
- chunks relevantes,
- contexto estruturado,
- taxonomia correta.

---

# 16. CONTEXT PRIORITY

A IA deve priorizar:

```text
System Rules
>
Structural Logic
>
Retrieved Knowledge
>
Narrative Layer
>
User Tone
```

---

# 17. PROMPT SEGMENTATION

Prompts devem ser:
- curtos,
- específicos,
- reutilizáveis,
- modulares.

---

diferentes.
