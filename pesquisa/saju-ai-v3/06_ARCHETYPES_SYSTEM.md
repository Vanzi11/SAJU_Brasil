# 06_ARCHETYPES_SYSTEM
## Sistema de Arquétipos Comportamentais

---

# VISÃO GERAL

O sistema de arquétipos existe para:
- aumentar identificação emocional,
- facilitar compreensão,
- gerar clareza narrativa,
- transformar estruturas complexas em perfis humanos memoráveis.

---

# OBJETIVO

Traduzir:
- elementos,
- Ten Gods,
- força do Self,
- padrões energéticos

em:
## perfis comportamentais intuitivos.

---

# REGRA CENTRAL

Arquétipos NÃO são:
- caixas fixas,
- personalidade absoluta,
- rótulos definitivos.

Arquétipos representam:
- tendências dominantes,
- estilo de funcionamento,
- padrões recorrentes.

---

# ESTRUTURA DO SISTEMA

Cada arquétipo deve conter:
- comportamento central,
- padrão emocional,
- estilo social,
- força principal,
- distorção principal,
- dinâmica profissional,
- frase-síntese.
