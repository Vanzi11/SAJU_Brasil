---

# 1. VISÃO GERAL

O sistema NÃO deve operar:
# como um único prompt gigante.

Isso gera:
- inconsistência,
- hallucination,
- mistura de lógica,
- perda de profundidade.

---

# 2. CORE PRINCIPLE

O sistema deve funcionar como:
# arquitetura multiagente especializada.

Cada agente:
- possui função específica,
- raciocínio delimitado,
- responsabilidade estrutural.

---

# 3. OBJETIVO

Criar:
- modularidade,
- profundidade,
- consistência,
- escalabilidade,
- precisão interpretativa.

---
